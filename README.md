# Chicken Road – Demo Casino Interface

## Overview
This project is a **demonstration interface** for a fictional online casino called **Chicken Road**.  
It was created to simulate how a basic casino platform could look and function (accounts, login, deposit page, etc.).

⚠️ **Important:** This is **not a real gambling platform** and **no real payments should be processed**.

The goal is purely **educational, experimental, and demonstrational**.
